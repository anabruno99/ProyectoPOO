Plantillas

Titulo

Definicion:

Son el mecanismo de C++ para implantar el paradigma de la programación genérica. Permiten que una clase o función trabaje con tipos de datos abstractos.

Plantillas de funciones: Es un modelo de función que el compiladro C++ usara para construir diferentes versiones de una misma función, según los tipos de datos que se especifiquen al invocar la misma.

Plantillas de clase: El sistema de plantillas nos permite definir una clase genérica que permita crear especializaciones o instancias de dicha plantilla para cualquier tipo especificado por un parámetro.
